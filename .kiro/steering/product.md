# Product Overview

Kliply is a professional macOS clipboard manager with Windows 11-style functionality. It provides clipboard history management with a keyboard-activated popup interface.

## Core Features

- **Clipboard History**: Tracks and stores clipboard items (text, rich text, images)
- **Keyboard Activation**: Quick access via configurable hotkey (default: Cmd+Shift+V)
- **Smart Deduplication**: Duplicate items move to front instead of creating new entries
- **Configurable Depth**: History size limit between 5-100 items (default: 10)
- **Content Type Support**: Plain text, rich text/HTML, and images

## Target Platform

macOS only - requires Accessibility permissions for global hotkey functionality.

## User Experience

- Minimal UI - activated on-demand via hotkey
- Windows 11-inspired design patterns
- First-launch welcome screen for permission setup
- Optional launch at login
